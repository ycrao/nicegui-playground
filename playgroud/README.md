playground
----------

>   My personal playground for [nicegui](https://github.com/zauberzeug/nicegui).


### markdown_render

Render markdown document.

```bash
cd markdown_render
uv run main.py
```

### rich_text_render

Render rich text (editor).

```bash
cd rich_text_render
uv run main.py
```

### table_render

Render a table component.

```bash
cd table_render
uv run main.py
```
